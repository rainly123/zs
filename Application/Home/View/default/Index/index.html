  • 登录进入企业招商界面
  • 注册和登陆成功

3.填写/提交信息及资料

  • 填写申请信息，提交资质

4.设置店铺名称和域名

  • 选择店铺名称及域名
  • 在线签署服务协议

5.等待第e商城审核

  • 第e商城7个工作日内给到审核结果

6.店铺上线，入驻成功
